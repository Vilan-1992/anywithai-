## Setup Instructions

### 1. Get Web3Forms API Key
1. Visit [https://web3forms.com](https://web3forms.com)
2. Sign up for a free account
3. Get your access key
4. Replace `YOUR_ACCESS_KEY_HERE` in index.html with your actual key

### 2. Deploy to GitHub Pages
This website is configured to work with GitHub Pages for free hosting.

### 3. Custom Domain Setup (GoDaddy)
After deploying to GitHub Pages, configure your GoDaddy domain:

1. Go to GoDaddy DNS Management
2. Add these DNS records:
   - Type: A, Name: @, Value: 185.199.108.153
   - Type: A, Name: @, Value: 185.199.109.153
   - Type: A, Name: @, Value: 185.199.110.153
   - Type: A, Name: @, Value: 185.199.111.153
   - Type: CNAME, Name: www, Value: yourusername.github.io

3. In your GitHub repository settings, add your custom domain
